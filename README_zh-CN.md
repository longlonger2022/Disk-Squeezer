## 更新日志

### Disk-Squeezer 2.0-Beta
- 有GUI界面, 用了Qt库。
- 比 2.0-Alpha 版稳定多了，可以使用写入模式，并且不会未响应。
- 命令模式会变得非常快。

### Disk-Squeezer 1.5
- 转成64位。
- 支持简体中文和English双语言切换。
- 增加“关于”选项。

### Disk-Squeezer 2.0-Alpha
- 增加了GUI界面。
- 极其不稳定，不建议使用。
- 速度极慢。

### 硬盘榨干器 1.4
- 增加核心功能：命令模式。

### 硬盘榨干器 1.3
- 增加了进度条。
- 增加了自定义显示的功能。

### 硬盘榨干器 1.2
- 只需要输入盘符，而不是完整路径。

### 硬盘榨干器 1.1
- 增加了自定义路径的功能。
